# Hacker-Tool

**Hacker-Tool** is an offline-first audit and automation toolkit with:

- A single-entrypoint CLI
- A full interactive launcher (TUI)
- A Termux-native manager (`htctl`)
- Network scanning with RFC1918 enforcement
- Filesystem, project, sync, and reporting modules

Use it on Termux (Android) or Linux.
